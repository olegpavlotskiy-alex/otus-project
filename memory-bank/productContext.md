# Product Context

## Проблема
Нет удобного инструмента для отслеживания личных расходов и доходов с визуализацией и аналитикой.

## Решение
Web-приложение с полноценным CRUD, дашбордом с графиками, поддержкой нескольких валют и автоматическим созданием повторяющихся платежей.

## Пользователи
- **John Smith** (`john@example.com` / `password123`) — личный аккаунт, USD, 197 транзакций за 6 мес.
- **Family Account** (`family@example.com` / `password123`) — семейный аккаунт, EUR

## User Flows

### Основной сценарий
1. Войти → попасть на Dashboard с графиками
2. Посмотреть баланс, расходы по категориям, динамику за 6 месяцев
3. Перейти в Transactions → добавить/редактировать транзакцию
4. Проверить Budgets — сколько потрачено от лимита в этом месяце
5. Настроить Recurring — автосоздание ежемесячных платежей

### Импорт данных
1. Transactions → Import CSV
2. Указать маппинг колонок (date, amount, type, category, description)
3. Получить результат: «X транзакций импортировано»
